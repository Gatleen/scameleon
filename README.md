# Scameleon

Scameleon is an anti-scam awareness web application designed to educate middle-aged adults about online scams, improve digital literacy, and promote safe online behavior through interactive learning.

The platform helps users recognize scam patterns, test their awareness, and understand what actions to take if they encounter or fall victim to scams.

---

## Features

- User authentication (Firebase Authentication)
- Educational content on common scam types through articles, videos and infographics
- Forum for users to discuss latest scams
- Diary for users to log their scam experiences
- Simulations for users to experience scams in a safe environment
- Interactive quizzes to test scam awareness
- Daily challenges and progress tracking
- Scam reporting guidance and emergency resources
- User profile with earned rewards and progress
- Fully responsive design
- More educational content

---

## Tech Stack

- **Frontend:** React + Vite
- **UI Library:** Chakra UI
- **Backend Services:** Firebase (Authentication & Firestore)
- **Hosting:** Vercel
- **Version Control:** Git & GitHub
